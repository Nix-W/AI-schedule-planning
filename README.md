# AI 日程规划系统

> 用自然语言描述，智能创建日程

**🌐 在线体验**：https://ai-schedule-planning-vtao.vercel.app/

---

## ✨ 功能特色

| 功能 | 说明 |
|------|------|
| 🗣️ **自然语言输入** | 像说话一样创建日程，无需手动填表 |
| 📅 **多视图日历** | 月视图、周视图、日视图自由切换 |
| 🔄 **重复日程** | 支持每天、每周、每月重复 |
| ⚠️ **冲突检测** | 自动检测时间冲突并提醒 |
| 🔍 **日程搜索** | 快速找到历史日程 |
| ⏰ **日程提醒** | 浏览器通知，不错过重要事项 |
| 🌙 **暗色模式** | 保护眼睛，夜间使用更舒适 |
| 📤 **导出日历** | 导出 .ics 文件，可导入系统日历 |
| ✏️ **拖拽编辑** | 拖拽调整日程时间 |

---

## 🚀 快速开始

### 创建日程

在输入框中用自然语言描述你的日程，然后点击「添加」：

```
明天下午3点开会
```

系统会自动解析并创建日程 ✨

### 支持的表达方式

**日期**
| 说法 | 效果 |
|------|------|
| 今天 | 当天 |
| 明天 / 后天 | 相对日期 |
| 下周一 | 下周的周一 |
| 下个月15号 | 下月15日 |
| 2月14号 | 指定日期 |

**时间**
| 说法 | 效果 |
|------|------|
| 上午 / 早上 | 9:00 |
| 中午 | 12:00 |
| 下午 | 14:00 |
| 晚上 | 19:00 |
| 3点 / 下午3点 | 15:00 |
| 3点半 | 15:30 |
| 8点到10点 | 时间范围 |

**更多信息**
| 说法 | 效果 |
|------|------|
| 在星巴克 | 添加地点 |
| 和老王 | 添加参与者 |
| 开会 / 会议 | 标记为会议类型（蓝色） |
| 休假 / 放假 | 标记为个人类型（紫色） |

**重复日程**
| 说法 | 效果 |
|------|------|
| 每天早上9点 | 每日重复 |
| 每周一 | 每周重复 |
| 每月15号 | 每月重复 |
| 工作日 | 周一至周五 |

### 示例

```
明天下午3点和老王在星巴克讨论项目
```
→ 📅 明天 15:00-16:00 | 📍 星巴克 | 👤 老王

```
每周一早上9点站会，30分钟
```
→ 📅 每周一 9:00-9:30 | 🔄 每周重复

```
下周五全天休假
```
→ 📅 下周五 全天 | 🏖️ 个人事项

## 📖 使用指南

### 查看日程

- 点击顶部的「月」「周」「日」切换视图
- 点击「今天」回到当前日期
- 点击「上一页」「下一页」翻页

### 编辑日程

1. 点击日历上的日程
2. 在弹窗中点击「编辑」
3. 修改标题、时间、地点等信息
4. 点击「保存」

或者直接**拖拽日程**到新的时间位置！

### 删除日程

1. 点击日历上的日程
2. 点击「删除」按钮
3. 确认删除

> 💡 删除重复日程时，可以选择只删除当前一条，或删除全部

### 搜索日程

1. 点击搜索图标或搜索框
2. 输入关键词（标题、地点、参与者）
3. 点击搜索结果跳转到该日程

### 设置提醒

1. 创建或编辑日程时
2. 选择提醒时间（5分钟前 / 15分钟前 / 30分钟前 / 1小时前）
3. 确保浏览器允许通知权限

> ⚠️ 首次使用时，浏览器会询问是否允许通知，请点击「允许」

### 导出日历

1. 点击右上角「导出日历」按钮
2. 下载 .ics 文件
3. 双击文件可导入到系统日历、Google Calendar、Outlook 等

### 切换主题

点击右上角的 🌙 / ☀️ 图标切换暗色/亮色模式

## 💡 小技巧

1. **使用快捷模板**：输入框下方有常用句式，点击即可填入

2. **预览确认**：系统解析后会先显示预览，确认无误再添加

3. **冲突提醒**：如果新日程与已有日程时间重叠，会显示警告

4. **数据保存**：所有日程自动保存在浏览器中，刷新不丢失

5. **批量管理**：重复日程可以一次性删除所有，或只删除某一条

---

## ⚠️ 注意事项

- 数据保存在浏览器本地，清除浏览器数据会丢失日程
- 换浏览器或设备需要重新添加日程
- 建议定期导出 .ics 文件备份
- 日程提醒需要浏览器保持打开状态
